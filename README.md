# 🎬 CineBot - Chatbot de Recomendação de Filmes

Projeto desenvolvido para a disciplina de **Qualidade de Software**, com foco em **usabilidade** e **confiabilidade**.

## 💡 Sobre o projeto

O **CineBot** é um chatbot interativo que recomenda filmes com base no gênero informado pelo usuário.

A aplicação simula uma conversa em tempo real, exibindo sugestões de filmes com pôster, título, ano e descrição.

---

## 🚀 Funcionalidades

- 💬 Interação em formato de chat
- 🎯 Recomendação de filmes por gênero
- 🎲 Sugestões aleatórias (3 filmes por busca)
- 🧠 Detecção de palavras-chave (ex: "ação", "terror", etc.)
- 🎭 Respostas dinâmicas e naturais
- ⌛ Simulação de "digitando..."
- 🖼️ Exibição de pôster + descrição dos filmes

---

## 🧪 Qualidade de Software

### ✅ Usabilidade
- Interface simples e intuitiva
- Feedback visual (digitando, respostas)
- Navegação fácil
- Interação natural

### ✅ Confiabilidade
- Tratamento de entradas inválidas
- Respostas padrão para erros
- Estrutura organizada e previsível
